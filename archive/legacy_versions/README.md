# 历史版本归档

当前开发版本是仓库根目录中的 V4.0.1。本目录仅保存旧版入口、旧单文件和迁移工具，避免它们与当前入口混放。

## 目录

- `v3_single_file/`：V3.9.3 单文件最终基线及历史命名启动器。
- `v4_0_preview/`：V4.0 模块化入口、启动器和一次性拆分工具。

归档中的 BAT 已离开原根目录，作为历史文件保留，不保证直接双击运行。恢复历史版本应优先使用下表对应的 Git 提交。GitHub 同时使用对应的小写版本标签，例如 `v3.9.3`、`v4.0` 和 `v4.0.1`。

## 恢复示例

查看 V3.9.3，但不改变当前分支：

```powershell
git worktree add ..\surface-rxy-v3.9.3 c86647e
```

恢复 V4.0：

```powershell
git worktree add ..\surface-rxy-v4.0 0fcfc7f
```

不要把历史文件复制回根目录覆盖 V4.0.1。
