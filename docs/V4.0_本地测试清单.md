# V4.0 本地测试清单

1. 双击 `start_surface_analyzer_v4_0.bat`，标题应显示 `V4.0`。
2. 导入一个过去能正常分析的小 XYZ/DAT，核对 Rx、Ry、PV、TTV 与 V3.9.3 一致。
3. 导入一个 Zeiss 大文件，核对默认文件位置抽样、导入点数和流畅度。
4. 导入白光分号 DAT 与带多行参数头的 XYZ，核对列映射和单位。
5. 分别测试矩形、圆形、智能抓面 ROI，清除当前/全部后视图应恢复。
6. 测试局部中位数和 Sigma 滤波，并导出主控报告图。
7. 测试多层扣减匹配诊断图。
8. 写入基准面和测量面，核对平行度与台阶结果、报告排版。
9. 导出再导入 Recipe，核对 ROI、滤波、大文件策略恢复。
10. 在命令行运行 `start_surface_analyzer_v4_0.bat --check`。

V4.0 尚未推到 GitHub。测试时发现问题可直接回退双击 `start_surface_analyzer_v3_9_3.bat`。
