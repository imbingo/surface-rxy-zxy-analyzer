# Surface Rxy ZXY Analyzer

面型及 Rxy 分析工具，当前主程序已更新到 V3.7.0。

## 文件

- `面型及Rxy分析ZXY版.py`: PyQt6 GUI 主程序，内容为 V3.7.0 最新版。
- `requirements.txt`: 运行所需 Python 依赖。

## 运行

```powershell
python -m pip install -r requirements.txt
python .\面型及Rxy分析ZXY版.py
```

## V3.7.0 重点（界面更新 · 对齐设计稿方案 A，纯 UI/样式，不改动任何算法）

- **四视图卡片化**：四幅图改为 2×2 四张独立卡片（白底圆角 + 柔和投影 + 留缝），模块感更强；
  子图标题改用 Qt 渲染的「● 标题」，蓝点与文字对齐。
- **模块化投影**：顶部读数卡、姿态磁贴、绘图卡均挂柔和投影，呈悬浮模块质感（车机 / Apple 风）。
- **顶部常驻读数条**：平均厚度 Z、面型 PV·法向、TTV·Z 极差、Rx、Ry 与平面方程上提为绘图区上方
  读数卡，淡底色 + 加大加粗数值强化读数，PV 卡蓝色高亮；去倾斜显示开关并入标题行，节省空间。
- **左栏分步引导**：工作流改为「1 载入 → 2 列映射 → 3 姿态 → 4 滤波」编号分步；姿态变换改为
  带线性图标的方块磁贴（↻ ↺ ⟳ ⇄ ↕ ↔ ⊕ ↶）。
- **统一中性风**：去掉全部 emoji 与多彩按钮，统一中性灰阶 + 单一强调色 `#2F6DB0`；自绘 chevron
  替换 Windows 原生下拉 / 微调箭头；删除 / 危险操作用低饱和红描边。
- **无边框自绘标题栏**：顶栏拖动移动 / 双击最大化、边缘拉伸缩放、最小化 / 最大化 / 关闭按钮。
- 说明：所有指标计算口径、数据流、滤波算法、Gap 运算、Recipe、批量处理、CSV 导出与报告图逻辑
  均保持不变；本版仅重排界面与样式。

## V3.6.0 重点

- **新增第 4 档滤波【迭代 σ 裁剪】**：反复用最佳拟合平面残差的标准差 σ 裁掉
  `|残差-均值| > Nσ` 的点并重拟合，直到残差 std 收敛或无新增剔除（单调裁剪，只剔不回收）。
  σ 倍数与迭代上限可配置（默认 3σ / 5 次）。适合“基本是平面 + 少量毛刺”的工件。
  注意：本档对全局残差判异，面有真实弧度时会削减 PV，弧形面建议优先用局部中位数滤波。
- **新增【批量处理】**：导入时可多选文件，沿用当前界面（或已导入 Recipe）的列映射 / 单位 /
  旋转 / 滤波设置逐个处理；每个文件输出一张含主页面全部信息（指标 + 四视图）的报告图
  `result_<原文件名>.png`，并汇总生成 `result_batch_summary.csv`。
  批量仅用自动滤波（无手动框选），要求所有文件为同一设备、同样列格式。
  建议先载入其中一个文件调好参数（或导入 Recipe），再点【批量处理】。

## 历史版本（V3.5.x）

- 未载入数据时切换去倾斜显示不再触发重绘异常。
- 局部中位数滤波改为分块近邻查询，降低大点云内存峰值。
- 支持超大 TXT / ASC / XYZ 文件按文件位置预抽样导入。
- 支持 Zeiss 文本常见缺测值如 `***`、`--`、`NA` 按空值处理。
- 导入状态会显示文件大小、导入方式、导入点数和显示点数。
- 绘图显示抽样上限可配置，指标按导入后的分析数据计算。
- Recipe 导出/导入可保存单位、列映射、旋转组合、滤波参数、显示和大文件策略。
